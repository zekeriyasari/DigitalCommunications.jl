DigitalCommunications.AWGNChannel — Type
mutable struct AWGNChannel{M}

Additive white Gaussian noise channel. AWGNChannel is defined by its snr. snr = Eb / No where Eb is the energy per bit and No/2 is the power spectral density of the channel noise. The relation between SNR, EsNo and EbNo is,

\[\begin{aligned} K * SNR &= EsNo \ EsNo &= k \times EbNo \ \end{aligned}\]

Fields

  • mode::Any

    Value of the noise mode(may be SNR, EsNo, EbNo in dB

  • k::Int64

    Bits per symbol

  • sps::Int64

    Samples per symbol

DigitalCommunications.PSK — Type
struct PSK{T} <: AbstractScheme

Phase Shift Keying. The mapping rule is

\[ s_m = g(t) cos(w_c t - \theta_m) \quad m = 1, \ldots, M\]

where $\theta_m = \dfrac{2\pi(m - 1)}{M}$.

source
DigitalCommunications.QAM — Type
struct QAM{T} <: AbstractScheme

Fields

  • M::Int64

  • alphabet::Any

Quadrature Amplitude Modulation. The mapping rule is

\[ s_m = A_{mi} g(t) cos(w_c t) - A_{mi} g(t) sin(w_c t) \]

where the amplitudes $A_{mi}, A_{mq} \in \{ \pm 1, \pm 3, \ldots, \pm (M - 1) \}$
